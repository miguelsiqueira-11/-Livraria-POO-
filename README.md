# Projeto Biblioteca Digital - Programação Orientada a Objetos

**Professor:** Daniel Bezerra

**Aluno:** Luiz Miguel Siqueira de Mendonça

## Descrição

Sistema de gerenciamento de uma biblioteca digital desenvolvido em Java para praticar conceitos de:

* Classes Abstratas e Herança
* Encapsulamento
* Polimorfismo
* Tratamento de Exceções Customizadas

## Como executar
1. Compile os arquivos: `javac *.java`
2. Execute: `java Main`
